# geo-fix — Подмена геолокации для доступа к Google-сервисам

## Что это

Программа для Windows, которая подменяет сигналы геолокации в браузере (часовой пояс, язык, GPS-координаты, WebRTC, DNS), чтобы в связке с VPN обеспечить доступ к NotebookLM, Gemini и другим гео-заблокированным сервисам Google.

**Что программа делает:**
- Подменяет часовой пояс в браузере (без изменения системного)
- Подменяет язык браузера и HTTP-заголовки
- Подменяет GPS-координаты (Geolocation API)
- Блокирует утечку IP через WebRTC
- Работает в Chrome, Edge и Firefox

**Что программа НЕ делает:**
- НЕ записывает и НЕ хранит ваш интернет-трафик
- НЕ отправляет данные куда-либо
- НЕ меняет системные настройки (всё возвращается после выключения)
- НЕ заменяет VPN — работает только вместе с ним

## Установка

### Вариант 1: Готовый архив (рекомендуется)
1. Скачайте `geo-fix-windows.zip` из [Releases](../../releases)
2. Распакуйте в любую папку (например, `C:\geo-fix\`)
3. Запустите `Включить (US).bat`
4. При первом запуске откроется мастер настройки

### Вариант 2: Автоустановщик
1. Скачайте весь проект
2. Запустите `install.bat` — он скачает Python, установит зависимости и создаст ярлыки автоматически

### Про сертификат

При первом запуске программа установит специальный сертификат. Это нужно для работы прокси, который подменяет геосигналы. Сертификат создаётся локально на вашем компьютере и никуда не отправляется.

### Про антивирус

Некоторые антивирусы могут показать предупреждение — это нормально. Программа использует mitmproxy (известный open-source инструмент), который антивирусы иногда принимают за угрозу. Добавьте папку `geo-fix` в исключения антивируса.

## Использование

### Через ярлыки на рабочем столе

- **Включить (US)** — маскировка под США
- **Включить (DE)** — маскировка под Германию
- **Включить (NL)** — маскировка под Нидерланды
- **Включить (GB)** — маскировка под Великобританию
- **Выключить** — остановить программу, вернуть всё как было

### Через командную строку

```
geo-fix.exe US        — включить маскировку (США)
geo-fix.exe DE        — включить маскировку (Германия)
geo-fix.exe --stop    — выключить
geo-fix.exe --cleanup — восстановить настройки после сбоя
geo-fix.exe --setup   — повторить первоначальную настройку
```

### Иконка в трее

Когда программа работает, в правом нижнем углу экрана (область уведомлений) появится иконка с кодом страны. Нажмите правой кнопкой мыши:

- **Активна: США (US)** — текущий статус
- **Переключить страну** → выбрать другую страну
- **Выключить** — остановить программу

## Как проверить, что всё работает

### Шаг 1: Включите VPN

Убедитесь, что VPN подключён и работает.

### Шаг 2: Запустите geo-fix

Дважды кликните ярлык нужной страны (например, **Включить (US)**).

### Шаг 3: Проверьте на CreepJS

1. Откройте в браузере: https://abrahamjuliot.github.io/creepjs/
2. Подождите 10-15 секунд, пока страница проанализирует ваш браузер
3. Что искать:
   - **Timezone**: должен быть `America/New_York` (для US) — НЕ ваш реальный часовой пояс
   - **Language**: должен быть `en-US` — НЕ `ru`
   - **Нигде не должно быть слова "Russia"**
   - Не должно быть красных предупреждений о несоответствиях

### Шаг 4: Проверьте WebRTC

1. Откройте: https://browserleaks.com/webrtc
2. Что искать:
   - Ваш реальный IP НЕ должен быть виден
   - Должен отображаться только IP вашего VPN

### Шаг 5: Проверьте DNS

1. Откройте: https://dnsleaktest.com
2. Нажмите "Extended test"
3. Что искать:
   - В результатах НЕ должно быть российских DNS-серверов
   - Серверы должны быть из страны вашего VPN

### Шаг 6: Откройте Google-сервисы

1. Откройте: https://notebooklm.google.com
2. Откройте: https://gemini.google.com
3. Что искать:
   - Страницы должны загружаться без ошибок
   - Не должно быть сообщения о недоступности в вашем регионе

**Если всё в порядке — маскировка работает!**

## Устранение неполадок

### Интернет перестал работать после сбоя

Если программа завершилась аварийно (компьютер выключился, программа зависла), прокси-настройки могли остаться. Решение:

1. Запустите ярлык **Выключить**
2. Или выполните: `geo-fix.exe --cleanup`
3. Или вручную: Настройки Windows → Сеть → Прокси → Выключить "Использовать прокси-сервер"

### VPN не обнаружен

Программа проверяет наличие VPN при запуске. Если VPN не обнаружен:
1. Убедитесь, что VPN подключён
2. Программа всё равно запустится, но IP может быть виден

### Некоторые сайты не работают

Сайты с особой защитой (банки, некоторые корпоративные порталы) могут не работать через прокси. Это нормально — выключите geo-fix для таких сайтов.

## Удаление

1. Запустите: `geo-fix.exe --cleanup` — удалит сертификат, правила файрвола, настройки прокси
2. Удалите папку `geo-fix`
3. Удалите ярлыки с рабочего стола

## Технические детали

- Работает через локальный HTTPS-прокси (mitmproxy) на порту 8080
- Прокси слушает только на localhost (127.0.0.1) — недоступен из сети
- Подмена сигналов через инъекцию JavaScript + модификацию HTTP-заголовков
- Все изменения обратимы — после выключения система возвращается в исходное состояние
- Исходный код: open-source, все зависимости бесплатные
