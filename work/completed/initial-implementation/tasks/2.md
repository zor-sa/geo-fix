# Task 2: JavaScript injection payload

## Description

Create the JS payload that overrides browser APIs for timezone, geolocation, language, and WebRTC. Must execute before page scripts via IIFE, use Object.defineProperty for stealth, dynamically compute timezone offset based on IANA zone name (DST-aware), and gracefully handle Temporal API absence in older browsers.

## Context Files
- `work/geo-fix/tech-spec.md` — D8 injection strategy
- `work/geo-fix/user-spec.md` — FR-1 signal list

## Acceptance Criteria
- [ ] IIFE wraps all override code, no global pollution
- [ ] `Date.prototype.getTimezoneOffset()` returns correct offset for configured timezone (DST-aware)
- [ ] `Intl.DateTimeFormat().resolvedOptions().timeZone` returns configured IANA zone
- [ ] `Temporal.Now.timeZoneId()` overridden if Temporal API exists, graceful skip if not
- [ ] `navigator.geolocation.getCurrentPosition()` returns configured coordinates
- [ ] `navigator.geolocation.watchPosition()` returns configured coordinates
- [ ] `navigator.language` and `navigator.languages` return configured language
- [ ] `RTCPeerConnection` constructor wrapped to block STUN server connections
- [ ] All overrides use `Object.defineProperty` with `configurable: false`
- [ ] Payload accepts configuration via placeholder tokens (TIMEZONE, LAT, LON, LANG) replaced by proxy addon

## TDD Anchor
- Test in Playwright/browser context: after injection, getTimezoneOffset returns expected value
- Test RTCPeerConnection creation is blocked or returns fake candidates
- Test navigator.language returns configured value
- Test geolocation returns configured coords
