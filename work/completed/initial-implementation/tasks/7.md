# Task 7: System tray icon

## Description

Create the tray icon module using pystray that shows active country, provides context menu for country switching and shutdown. Country switching must be thread-safe (synchronized with proxy addon). Tray icon must update when country changes and cleanly remove on exit.

## Context Files
- `src/presets.py` — country presets
- `work/geo-fix/user-spec.md` — FR-4 tray menu specification

## Acceptance Criteria
- [ ] Tray icon visible when app is running
- [ ] Icon shows country abbreviation (text-based icon via Pillow)
- [ ] Right-click menu: status line (disabled), separator, country submenu (US/DE/NL/GB), separator, "Выключить"
- [ ] Country switching via submenu updates icon and triggers callback
- [ ] "Выключить" triggers clean shutdown callback
- [ ] Country switch callback is thread-safe (uses threading.Lock)
- [ ] Icon cleanly removed on exit
- [ ] Menu text in Russian

## TDD Anchor
- Test: menu items generated correctly for all 4 presets
- Test: country switch callback invoked with correct preset
- Test: shutdown callback invoked on "Выключить"
