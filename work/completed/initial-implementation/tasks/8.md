# Task 8: Main entry point and CLI

## Description

Create the main orchestrator that validates CLI arguments, acquires single-instance mutex, runs crash recovery if needed, runs setup wizard if first time, starts mitmproxy with addon on 127.0.0.1:8080, configures system proxy, launches tray icon, and handles clean shutdown via atexit handler. Supports --stop and --cleanup commands.

## Context Files
- All `src/*.py` modules
- `work/geo-fix/tech-spec.md` — D9 crash recovery, D10 proxy binding

## Acceptance Criteria
- [ ] CLI: `geo-fix US` starts with US preset, `geo-fix --stop` stops, `geo-fix --cleanup` cleans up
- [ ] Country code validated: uppercase 2-letter, must exist in PRESETS
- [ ] Invalid input → clear error message, exit before any system changes
- [ ] Single-instance guard checked before any system changes
- [ ] Stale state file detected → cleanup runs before new session
- [ ] Setup wizard runs on first launch
- [ ] mitmproxy started on 127.0.0.1:8080 (not 0.0.0.0)
- [ ] Startup assertion verifies proxy bound to localhost
- [ ] System proxy set (WinINET + Firefox) after proxy is running
- [ ] Tray icon started after proxy is running
- [ ] atexit handler registered for cleanup
- [ ] --stop: sends stop signal to running instance (via state file PID)
- [ ] --cleanup: runs cleanup without starting a new session
- [ ] Clean shutdown: proxy reverted, state file deleted, mitmproxy stopped, tray removed
