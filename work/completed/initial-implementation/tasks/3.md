# Task 3: mitmproxy addon

## Description

Create the mitmproxy addon that rewrites Accept-Language headers on all requests, and for target domains only: injects JS payload with nonce into HTML responses and modifies CSP to allow the nonce. Must handle gzip/brotli encoding, use HTML parser for injection point, skip non-HTML and large responses (>5MB), and pass through non-target traffic unmodified.

## Context Files
- `src/inject.js` — JS payload template
- `src/presets.py` — country presets and target domains
- `work/geo-fix/tech-spec.md` — D8 CSP nonce strategy, D10 proxy binding

## Acceptance Criteria
- [ ] Accept-Language header rewritten on ALL requests to match active preset
- [ ] JS payload injected into HTML responses from target domains only
- [ ] Nonce generated per-response, added to both script tag and CSP script-src
- [ ] Other CSP directives (frame-ancestors, form-action, etc.) preserved
- [ ] Non-target domain traffic passes through with original CSP intact
- [ ] Non-HTML responses (images, JSON, etc.) pass through unmodified
- [ ] Responses >5MB skipped for injection
- [ ] gzip/brotli encoded responses handled correctly
- [ ] HTML parser used for injection point (not regex)
- [ ] Country preset switchable at runtime (thread-safe reference)

## TDD Anchor
- Test: send HTML request to google.com domain, verify response contains injected JS with nonce
- Test: send HTML request to non-target domain, verify response unmodified
- Test: send request, verify Accept-Language header matches preset
- Test: verify CSP header has nonce appended to script-src, other directives intact
- Test: send >5MB HTML response, verify no injection
- Test: send non-HTML response, verify passthrough
