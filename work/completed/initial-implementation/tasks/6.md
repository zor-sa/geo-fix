# Task 6: Setup wizard

## Description

Create a first-run setup wizard using tkinter that guides the user through CA certificate installation (Chrome/Edge + Firefox), optional firewall rules, and DNS configuration for each browser. Must show clear explanations in Russian, handle errors gracefully, and persist setup completion state.

## Context Files
- `src/system_config.py` — cert install, firewall, proxy config functions
- `src/health_check.py` — status checks
- `work/geo-fix/tech-spec.md` — D4 DNS guide, D5 wizard design

## Acceptance Criteria
- [ ] Wizard shown only on first run (completion state persisted)
- [ ] Step 1: CA certificate install explanation (Russian) + install button
- [ ] Step 2: Firefox CA trust configuration
- [ ] Step 3: Optional firewall rules (checkbox, explains admin will be needed)
- [ ] Step 4: DNS setup guide with "open Chrome settings" and "open Firefox settings" buttons
- [ ] Each step shows success/failure status
- [ ] Error messages in Russian, human-readable
- [ ] Completion flag prevents re-run on subsequent launches
- [ ] Wizard can be re-triggered with --setup CLI flag

## TDD Anchor
- Test: completion flag written after successful wizard
- Test: wizard not shown when completion flag exists
- Test: wizard shown when --setup flag used regardless of completion state
